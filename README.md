# Hospital Appointment System (Django REST API)

# Project Overview
A fully functional Hospital Appointment System built with Django and Django REST Framework, allowing patients to book appointments with doctors, doctors to manage their schedules, and admin to manage all users.

# Key Features:

1.Custom User model with roles (patient, doctor, admin)

2.Patient and Doctor profiles with CRUD operations

3.Appointment booking and management

4.Medical records management

5.JWT authentication for secure access

6.Fully RESTful API endpoints

7.Ready for Postman testing or frontend integration
